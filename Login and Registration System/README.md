🌟 Login & Registration System (C++)
A simple and educational login & registration system built in C++ that stores user data using files. Perfect for learning file handling, basic hashing, and terminal-based programs.

----

🚀 Features

✅ Create new accounts with username & password
✅ Login with existing accounts
✅ Username validation: Only letters, digits, and underscores allowed
✅ Password validation: Minimum 8 characters
✅ Cross-platform folder creation (Windows, Linux, macOS)
✅ Colored terminal output for better user experience

---

📋 Requirements

C++17 or later
Works on Windows, Linux, and macOS terminals

💡 Learning Goals

Learn file handling in C++
Understand basic user authentication systems
Practice input validation and terminal UI design
Explore cross-platform folder handling

---

⚠️ Limitations

Passwords are not cryptographically secure
No encryption of user data
Only supports single user login sessions at a time
